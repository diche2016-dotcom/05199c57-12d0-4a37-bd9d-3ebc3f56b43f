Bookstore WordPress Installation Package
----------------------------------------

Contents:
1. Theme: bookstore_theme/
2. Plugins: plugins/
3. README.txt: This instruction file

Installation Steps:
1. Extract this ZIP archive.
2. Upload 'bookstore_theme' folder to /wp-content/themes/ on your server.
3. Upload all plugins inside 'plugins' folder to /wp-content/plugins/ on your server.
4. Activate the theme and plugins from the WordPress admin dashboard.
